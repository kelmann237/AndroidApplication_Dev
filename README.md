A Flutter mobile application for managing and calculating student grades. Supports single student entry, bulk Excel import, grade report generation, and export as PDF or TXT.

 Features

Single Student Entry: Enter a student's name, registration number, course, major, and mark to instantly compute their grade and remark
Excel Batch Import: Upload a .xlsx file to import an entire class at once; results are sorted by mark with live statistics
Automatic Grading : Marks are automatically converted to letter grades (A+ through F) with descriptive remarks
Save & Review : Save individual or batch results and review them at any time from the Saved Records page
Export Reports: Export any grade report as a formatted .pdf or plain .txt file and share it via any app on your device
Copy to Clipboard : One-tap copy of any report in plain text format
